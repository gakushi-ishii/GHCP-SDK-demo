# スクリーンショット

このディレクトリには、デモアプリケーションのスクリーンショットが含まれています。

## メインメニュー

`main-menu.txt` - アプリケーション起動時のメインメニュー画面

実際に `npm run dev` を実行すると、対話型メニューから各デモを選択できます。

## 実行方法

```bash
# デモアプリを起動
npm run dev

# または個別のデモを実行
npm run demo:chat
npm run demo:code-gen
npm run demo:context
```

詳細な実行例については、`docs/EXAMPLES.md` を参照してください。
